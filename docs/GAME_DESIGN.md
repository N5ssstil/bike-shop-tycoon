# 公路车模拟经营游戏 - 设计文档

> 项目代号：Bike Shop Tycoon（暂定）
> 目标玩家：硬核公路车爱好者
> 美术风格：写实风
> 开发模式：AI 开发 + AI Review + 用户提供美术资源

---

## 一、游戏概述

### 核心理念
从街边自行车店做起，逐步成长为职业车队老板，最终冲击环法冠军。

### 核心循环
```
进货 → 卖车/修车 → 赚钱 → 升级店铺 → 解锁高级内容 → 组建车队 → 参加赛事 → 环法冠军
```

### 三大爽点
1. **即时反馈**：每次操作都有明确正向反馈（赚差价、提口碑、清库存）
2. **成长解锁**：从入门铝架到顶级碳车，从小店到大店
3. **情感连接**：顾客故事、骑友圈子、车手成长

## 三、第一阶段：核心原型（1-2个月）

### 3.1 核心目标
完成车店基础经营玩法，实现「进货-卖车-修车」核心循环，打造最简可玩原型。

### 3.2 开发内容

#### 3.2.1 基础界面
- 店铺初始场景（狭小老店铺）
- 主菜单
- 商品展示界面
- 顾客接待界面
- 简单维修界面

#### 3.2.2 核心玩法

**进货与库存**
- 对接基础中小品牌厂商
- 进货、库存管理（查看库存、补货、清理滞销品）

**顾客接待**
- 3 类基础顾客：
  | 顾客类型 | 需求特点 | 推荐车型 |
  |----------|----------|----------|
  | 学生 | 预算有限、追求性价比、可能备战校园赛 | 入门铝合金公路车 |
  | 通勤族 | 耐用、舒适、日常代步 | 耐用通勤车/入门公路车 |
  | 普通骑行爱好者 | 有一定预算、追求性能/颜值 | 中端公路车 |
- 对话、需求识别、车型推荐
- 完成交易流程

**基础维修**
- 爆胎更换
- 变速调试
- 刹车保养
- 简单操作流程（点击工具→完成维修）

#### 3.2.3 基础数值体系
- 商品定价、进货成本
- 维修费用
- 口碑值（影响客流）

#### 3.2.4 里程碑目标（新增）
| 里程碑 | 条件 | 奖励 |
|--------|------|------|
| 第一桶金 | 累计盈利 1 万 | 解锁「品牌代理资格」提示 |
| 小有名气 | 口碑值达到 50 | 客流增加 10% |
| 第一桶金（进阶） | 累计盈利 10 万 | 解锁第一个知名品牌代理权 |

#### 3.2.5 顾客故事系统（情感连接）
- 学生：攒钱买第一台车备战校园赛
- 通勤族：换车是为了每天多陪孩子
- 骑行爱好者：来修车时分享骑游经历

### 3.3 验收标准
- 可正常完成「进货-接待顾客-卖车/修车」全流程
- 界面切换流畅，操作逻辑易懂
- 库存、口碑值可正常变化

---

## 四、第二阶段：车店进阶（2-3个月）

### 4.1 核心目标
完善车店升级、商品体系、进阶服务，增加经营深度。

### 4.2 开发内容

#### 4.2.1 店铺升级系统
- 扩大门面
- 解锁展示区、维修区、改装工作室
- 升级所需资源（资金、口碑）

#### 4.2.2 商品体系完善
- 竞赛级公路车
- 高端零件（禧玛诺、速联套件）
- 骑行周边（水壶、骑行台）
- 解锁品牌代理权、议价功能

#### 4.2.3 进阶服务
- 车辆改装（换轮组、改涂装）
- 定制服务（指定涂装、调整车架尺寸）

#### 4.2.4 顾客体系完善
- 竞赛车手：顶级配置需求
- 网红/拍照党：颜值涂装需求
- 增加顾客互动剧情

#### 4.2.5 社群基础
- 添加顾客好友
- 发布骑游通知
- 积累社群粉丝

---

## 五、第三阶段：社群与车队基础（2-3个月）

### 5.1 核心目标
完善社群与活动玩法，解锁车队组建权限。

### 5.2 开发内容

#### 5.2.1 社群与活动
- 周末骑游、新手教学、保养课程
- 本地绕圈赛、爬坡赛
- 赛事报名、简单赛程展示

#### 5.2.2 车队组建基础
- 解锁条件：店铺等级 + 口碑 + 现金流
- 车手招募：从顾客、赛事参与者中筛选
- 车手分类：新人、业余赛手

#### 5.2.3 车手属性系统（新增）
参考《足球经理》球员属性：

| 属性 | 说明 |
|------|------|
| 爬坡 | 爬坡能力，影响山地赛段表现 |
| 冲刺 | 终点冲刺能力 |
| 计时赛 | 个人计时赛能力 |
| 耐力 | 长距离赛事的体力保持 |
| 恢复力 | 赛后恢复速度 |
| 战术意识 | 执行战术的效果 |
| 团队协作 | 配合队友的能力 |

#### 5.2.4 车手性格系统（新增）
| 性格类型 | 特点 |
|----------|------|
| 好胜型 | 关键时刻有加成，但可能不听战术 |
| 团队型 | 默默配合队友，副将首选 |
| 关键时刻爆发型 | 大赛表现更好 |
| 稳定型 | 表现稳定，少出错 |

#### 5.2.5 车店与车队联动
- 店铺为车队提供战车、维修保养
- 车队参与本地活动提升店铺口碑

#### 5.2.6 随机事件
- 暴雨客流暴跌
- 供货商涨价
- 网红顾客带来客流（新增）

---

## 六、第四阶段：车队进阶与赛事（3-4个月）

### 6.1 核心目标
完善车队经营、赛事战术、战车定制功能，实现从本地赛事到环法的完整进阶。

### 6.2 开发内容

#### 6.2.1 车队经营完善
- 车手类型：职业赛手、退役老将
- 车手角色：爬坡手、冲刺手、副将、全能主将
- 团队配置：教练、机械师、营养师
- 车手转会、伤病机制

#### 6.2.2 赛事体系
| 赛事等级 | 示例 |
|----------|------|
| 初级 | 本地绕圈赛、爬坡赛 |
| 中级 | 省级联赛、全国业余锦标赛 |
| 高级 | 职业洲际赛、环中国 |
| 终极 | 环法自行车赛 |

#### 6.2.3 赛事模拟方式
**纯数值模拟**：
1. 设置战术（副将领骑、主将保存体力、冲刺绝杀等）
2. 根据赛道特点调整
3. 观看结果播报

#### 6.2.4 环法细节
- 还原真实赛段（平路、爬坡、计时赛）
- 黄衫、绿衫、波尔卡圆点衫争夺
- 与顶级 AI 车队同台竞技

#### 6.2.5 改装与造车进阶
- 车队专属定制战车
- 调校参数（平路气动、爬坡轻量化）
- 自有车架研发

---

## 七、第五阶段：终极完善与上线（2-3个月）

### 7.1 核心目标
完善全功能，修复 BUG，完成上线准备。

### 7.2 开发内容

#### 7.2.1 终极内容
- 全国连锁店铺
- 自有车架生产工厂
- 品牌研发中心
- 世界顶级车队解锁
- 环法总冠军争夺

#### 7.2.2 商业化
- 外观付费（车架涂装、骑行服、车队队服）
- 内容解锁（专属赛道、定制车架模板）
- 赛季通行证（无强制付费、无垃圾抽卡）

#### 7.2.3 上线准备
- 内部测试、公开测试
- 宣传素材
- 平台上架
