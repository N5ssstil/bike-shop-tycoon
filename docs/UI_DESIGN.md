# UI 布局需求文档

> 分辨率：1920×1080
> 风格：写实风，现代简洁，带有公路车运动感

---

## 一、设计原则

1. **清晰易读**：信息层次分明，重点突出
2. **操作直观**：常用功能一键可达
3. **风格统一**：整体色调、字体、图标风格一致
4. **公路车元素**：融入链条、齿轮、碳纤维纹理等元素

---

## 二、色彩方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | 深灰/碳黑 | #1A1A1A |
| 次色 | 骑行绿 | #2ECC71 |
| 强调色 | 警示橙 | #E67E22 |
| 背景色 | 浅灰 | #F5F5F5 |
| 文字色 | 深灰 | #333333 |
| 金钱色 | 金色 | #F1C40F |
| 口碑色 | 蓝色 | #3498DB |

---

## 三、字体建议

| 用途 | 字体风格 | 示例 |
|------|----------|------|
| 标题 | 粗体无衬线 | Montserrat Bold |
| 正文 | 清晰易读 | Roboto Regular |
| 数字/金额 | 等宽字体 | Roboto Mono |
| 强调 | 中等粗细 | Roboto Medium |
